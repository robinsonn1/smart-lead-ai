# 🚀 Smart Lead AI

AI-powered lead qualification API built with FastAPI and OpenAI.

## 🔥 Features
- Extracts structured data from customer inquiries
- Classifies:
  - intent
  - budget
  - urgency
  - location
- Generates lead score (1–10)
- Combines LLM + rule-based logic

## 🧠 Tech Stack
- Python
- FastAPI
- OpenAI API
- Prompt Engineering
